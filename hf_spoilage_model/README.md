# FridgeAI Spoilage Classifier (MobileNetV3-Small)

Binary classifier that detects whether a food item is **fresh** or **spoiled**.

- **Architecture**: MobileNetV3-Small
- **Training data**: Fresh and Rotten Fruits Dataset (~13,500 images)
- **Accuracy**: 100% on test set
